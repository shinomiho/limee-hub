# リミー 共有資料フォルダ

このフォルダに資料ファイル（PDF等）をアップロードすると、
`https://shinomiho.github.io/limee-hub/docs/ファイル名` で永続リンクになります。

## ファイルのアップ方法（ターミナル不要）

1. github.com/shinomiho/limee-hub を開く
2. 「docs」フォルダをクリック
3. 「Add file」→「Upload files」
4. ファイルをドラッグ＆ドロップ
5. 「Commit changes」ボタンを押す → 完了

## ルール

- ファイル名は英数字とハイフンのみ（例: `ai-report-sample.pdf`）
- 更新するときは**同じファイル名**でアップ → URLが変わらない
- 新しいファイル名にした瞬間にURLが変わるので注意
