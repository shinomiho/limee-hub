🏢 リミーとは何か

リミー（Rimee）は、採用後支援・オンボーディング特化のHRテック企業です。代表は篠辺美穂（Miho Shinohe）。

主なサービスは2つ：
① 入社90日AI診断システム：新入社員が25問回答 → AIがレポート生成 → 管理者へ自動送信 → 人的資本KPIとして開示支援
② 人的資本経営のKPI設計支援：インプット・プロセス・アウトカムの3層構造でKPI設計

ビジョン：「AIと人の力で人事の未来を変える」

🛠️ システムの更新・メンテナンス方法
🤖
Claude Codeを使った更新（推奨）

リミーのシステムはすべてClaude Code（AIアシスタント）を使って構築・更新しています。コードの知識がなくても、日本語で指示するだけで更新できます。

  • 1️⃣ターミナルを開く（Mac：Command+Space → 「ターミナル」）
  • 2️⃣claude と入力してEnter → Claude Codeが起動
  • 3️⃣「〇〇を変更したい」「〇〇が壊れた」など日本語で伝える
  • 4️⃣Claudeが修正してくれる → 内容を確認してGASに貼り付け
📊
AI診断システムの更新手順
  • 1️⃣Claude Codeで cd /Users/mihomatuo/onboarding-diagnosis と入力
  • 2️⃣変更したい内容を日本語で伝える（例：「診断の質問3番を変更したい」）
  • 3️⃣HTMLファイルの変更 → GitHubにpush → 自動でサイト更新
  • 4️⃣GASの変更 → Claudeがコードをコピー → GASエディタに貼り付け → 保存・デプロイ
📱
SNS自動投稿の更新手順
  • 1️⃣Claude Codeで cd /Users/mihomatuo/social-auto-post と入力
  • 2️⃣変更内容を伝える（例：「投稿文のトーンを変えたい」「新しいSNSを追加したい」）
  • 3️⃣Claudeがapps-script.gsを修正 → クリップボードにコピー
  • 4️⃣SNS自動投稿GASを開く → Ctrl+A → Ctrl+V → 保存
🗺️
このhubページの更新手順
  • 1️⃣Claude Codeで cd /Users/mihomatuo/limee-hub と入力
  • 2️⃣「〇〇のリンクを追加して」「〇〇のステータスを変更して」など伝える
  • 3️⃣Claudeがindex.htmlを修正 → GitHubにpush → 自動で反映

  • 2️⃣Stand.FM録音・投稿：台本はカレンダーに自動保存。読んで録音するだけ。
  • 3️⃣顧客対応メール：テンプレートあり。契約後の企業サポート。
  • 4️⃣システムの軽微な更新：Claude Codeに日本語で指示するだけで対応可能。
  • 5️⃣KPIレポート確認：管理画面のKPIタブを月1回確認して報告。
🚨 緊急時・困ったときの対処法
🤖
システムが壊れた・動かない
Claude Codeを起動して「〇〇が動かない」と伝える。エラーメッセージをそのまま貼り付ければ原因を特定して直してくれる。
🔑
APIキーが切れた・エラーが出る
各サービスのダッシュボードで新しいAPIキーを発行 → GASの「プロジェクトの設定」→「スクリプトプロパティ」で更新する。
📧
エラーメールが届いた
件名に「SNSエラー」または「Script function not found」とあればGASの問題。Claude Codeにエラー内容を貼り付けて対処してもらう。
💳
課金・サブスクの確認
Anthropic・ElevenLabs・Pexels・Google Cloudの料金が発生している。月次で確認し、不要なら解約・ダウングレードを検討する。
